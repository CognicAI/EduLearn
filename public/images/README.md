# EduLearn Images Directory

This directory contains static images for the EduLearn platform.

## Directory Structure:
- `/og/` - Open Graph images for social media sharing
- `/logos/` - Brand logos and icons
- `/avatars/` - Default user avatars
- `/backgrounds/` - Background images
- `/course-thumbnails/` - Course preview images

## Image Guidelines:
- **Open Graph Images**: 1200x630px (recommended)
- **Logos**: SVG preferred, PNG fallback
- **Avatars**: 200x200px minimum
- **Course Thumbnails**: 400x300px

## Supported Formats:
- PNG, JPG, WEBP, SVG
- Use WEBP for better compression when possible
