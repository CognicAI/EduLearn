How to create PNG for social media:

  1. Right-click on the image above
  2. Select "Inspect Element" or "Inspect"
  3. In the developer tools, right-click on the element with class "og-image"
  4. Select "Capture node screenshot" or similar option
  5. Save the image as "og-default.png"
  6. Upload it to your server at "/public/images/og/og-default.png"

Alternative method:

  1. Take a screenshot of the blue image above
  2. Crop it to exactly 1200x630 pixels
  3. Save as PNG format
  4. Upload to your server

For best results: Use tools like Canva or Figma to create a professional 1200x630px PNG image.
